## GO - C.R.U.D. - MARIADB 

Before you start using this CRUD, you need to create a mariadb database 
and set: "user", "password", "host", "port" and "db_name", 
save these environment variables in an ".env" file in the root project.


 .env file format 
```
user="<db_user>"
pass="db_pass"
host="<db_host>"
port="3306"
db_name="<db_name>"
```
